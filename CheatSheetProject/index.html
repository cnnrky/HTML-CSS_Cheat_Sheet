CSS Selectors and Specificity

Types of Selectors

Name HTML CSS Order of Specificity
Universal None * {style} Least Specific
Type <h1>Heading 1</h1> h1 {style} Less Specific
Class <h1 class='guitar'>Heading 1</h1> .guitar {style} More Specific
ID <h1 id='telecaster'>Heading 1</h1> #telecaster {style} Most Specific

Using Multiple Selectors

Name CSS Description Usage
Selector List h1, h2, h3 {style} List mutliple selectors separated by commas-
can be separated onto individual lines		 Applies styles to all listed elements
Descendant Combinator h1 li {style} List multiple selectors with space between Applies style only to list items that are descendants of h1 elements
Chaining h1.guitar List multiple selectors for no space between Applies style to h1 elements that also have the 'guitar' class
